A simple and modern Movie App built using HTML, Tailwind CSS, and JavaScript, powered by the TMDB (The Movie Database) API. This app allows users to browse, search, and explore movies with a clean and responsive UI.

🚀 Features
🔍 Search for movies in real-time
🎞️ View popular and trending movies
🖼️ Display movie posters, titles, and ratings
📄 View detailed movie information
▶️ Watch trailers (via YouTube integration)
📱 Fully responsive design (mobile-friendly)
🎨 Styled with Tailwind CSS
🛠️ Technologies Used
HTML5 – Structure of the app
Tailwind CSS – Styling and layout
JavaScript (ES6+) – Functionality and API handling
TMDB API – Movie data source
